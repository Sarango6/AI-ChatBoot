# AI Chatbot 🤖

## Description
An Artificial Intelligence chatbot built using Python and NLP techniques.

## Features

- Text based conversation
- NLP similarity matching
- Chat interface
- Multiple responses
- Chat history

## Technology Used

- Python
- Streamlit
- Scikit Learn
- NLP


## Run Project

Install dependencies

pip install -r requirements.txt


Run:

streamlit run app.py
